> 致：长城人寿  
> 更新日期：2026-07-13

## 一、产品定位

**一句话定位：**
> 代理人专属的AI营销全能助手，覆盖"见面前→见面中→见面上→日常经营"全场景，让每个代理人都拥有一个随身AI陪访专家。

**核心价值：**
| 角色 | 价值 |
|------|------|
| 对代理人 | 少跑冤枉路、多说关键话、多签优质单 |
| 对团队主管 | 陪访效率提升、新人成长加速、优秀经验可复制 |
| 对公司 | 展业标准化、合规可控、数据资产沉淀 |

## 二、目标用户画像

| 维度 | 说明 |
|------|------|
| 用户 | 长城人寿个险渠道代理人 |

### 🔴 核心业务线

#### 模块1：智能拜访全流程管理

**拜访前（准备）**
| 功能 | 一句话描述 |
|------|-----------|
| 今日客户推荐 | 基于知客系统客户画像，智能排序今日最佳拜访对象 |
| 路线规划 | 一键规划最优拜访路线，显示预计耗时 |
| 沟通议题推荐 | 根据客户经营阶段，自动推荐本次沟通要点 |
| 物料一键准备 | 生成客户定制建议书 + 物料清单 |

**拜访中（执行）**
| 功能 | 一句话描述 |
|------|-----------|
| 语音实时转写 | 自动录音转文字，关键信息自动提取 |
| 智能话术推荐 | 根据客户画像+实时对话，推荐最佳应答话术 |
| 知识库问答 | 产品条款、投保规则、公司服务，随时问 |
| 方案展示 | 手机上直接展示建议书、收益演示 |

**拜访后（跟进）**
| 功能 | 一句话描述 |
|------|-----------|
| 智能小结 | 语音记录→自动生成结构化拜访记录 |
| 跟进待办 | 自动生成24h感谢联络、48h资料补发等任务 |
| 数据归档 | 拜访记录自动同步至客户档案 |

#### 模块2：智能化陪访管理

| 功能 | 一句话描述 |
|------|-----------|
| 陪访对象筛选 | 自动识别需要陪访的高潜力/复杂客户 |
| 人员匹配 | 根据客户类型推荐最合适的陪访人员 |
| 现场协同 | 主谈/辅谈角色分工，应急方案 |
| 过程记录 | 实时记录客户反馈，标记亮点和不足 |
| 效果复盘 | 成单率统计，归因分析，代理人短板诊断 |
| 案例沉淀 | 优秀陪访案例自动入库，全员可学 |

#### 模块3：智能建议书生成

| 功能 | 一句话描述 |
|------|-----------|
| 需求预判 | 根据客户信息自动判断保障缺口和理财需求 |
| 方案生成 | 一键生成定制保障方案 |
| 灵活调整 | 保额、缴费期、保障责任随便调 |
| 可视化展示 | 责任对比图、收益演示、多方案并排对比 |
| 一键导出 | PDF/微信分享，随时发给客户 |

## 四、分阶段实施建议

```
阶段一（Demo/PoC）  →  阶段二（MVP）  →  阶段三（全面铺开）
    1-2个月              3-4个月             持续迭代
  ┌────────────┐      ┌────────────┐      ┌────────────┐
  │ 拜访管理    │      │ + 陪访管理  │      │ + 日程管家  │
  │ + 建议书    │      │ + 客户经营  │      │ + 合规风控  │
  │ （核心故事） │      │ （丰富功能） │      │ （完整平台） │
  └────────────┘      └────────────┘      └────────────┘
```

### 阶段一：Demo/PoC（搞定立项）
**核心模块：**
- 智能拜访全流程（拜访前→中→后，完整闭环）
- 智能建议书生成（从需求到方案一键出）

**目标：** 用一条完整故事线打动客户，确认立项

### 阶段二：MVP（快速上线）
**新增模块：**
- 智能化陪访管理
- 客户智能化经营
- 数据同步对接知客系统

**目标：** 代理人能用起来，跑通真实业务场景

### 阶段三：全面铺开
**新增模块：**
- 日程智能管家
- 合规与风控管理
- 更多AI能力（深度个性化推荐、预测分析等）

**目标：** 成为代理人离不开的展业工具

### AI能力体系
| 能力 | 方案建议 |
|------|----------|
| 大语言模型（对话/话术/总结） | 通义千问 / 文心一言 / 开源模型 |

## 七、Demo故事线（后续做Demo时的脚本框架）

### 主角：代理人小王

### 场景一：早晨 8:00 — 开工
> 小王打开手机，看到今日早报——今天有3个客户待拜访，2个待跟进，1个待陪访。
> 系统自动推荐了最优拜访路线：先去张先生家（小区A），然后李女士公司（同区），最后王总办公室（顺路）。

### 场景二：上午 9:30 — 拜访中
> 小王到了张先生家，打开语音转写，对话自动记录。
> 张先生问："这个重疾险和医疗险有啥区别？"小王点开智能问答，秒回。
> 聊到方案时，一键调出定制建议书，在手机上直接展示收益演示。

### 场景三：上午 11:00 — 拜访结束
> 点击"结束拜访"，系统自动生成拜访小结和跟进任务。
> 今晚发感谢短信，明天补发详细资料——系统都安排好了。

### 场景四：下午 14:00 — 陪访
> 主管老李收到陪访推送，小王有个复杂客户需要支持。
> 老李查看客户信息后一键确认陪访，系统自动生成陪访简报。
> 陪访结束后，老李的点评和建议自动归档为小王的成长档案。

### 场景五：傍晚 18:00 — 收工
> 小王查看今天的拜访数据：跑了3家，签了1单，收集了2个跟进线索。
> 系统推送明天的最佳拜访对象——今天表现不错！
